## 1. 设计原则与风格概述

### 1.1 设计理念

网站采用**沉浸式娱乐媒体风格**，以深色模式为基调，营造影院级视觉体验。设计核心原则：

- **内容为王**：深色背景最大化突出彩色内容图片，让视觉焦点始终集中在谢可寅的作品和形象上
- **金色尊贵**：金色强调色呼应VIP专题页质感，传递艺人品牌的高级感
- **沉浸流畅**：全屏Hero、平滑过渡、微动效，打造沉浸式浏览体验
- **信息清晰**：高对比度文字、明确的信息层级、直观的交互反馈

### 1.2 风格关键词

```
深色沉浸 · 金色尊贵 · 影院质感 · 现代简洁 · 娱乐媒体
```

### 1.3 参考对标

- 爱奇艺/腾讯视频个人专题页
- Netflix 个人资料页
- Apple TV+ 艺人专题页
- 深色模式媒体展示平台

## 7. 动画与动效规范

### 7.1 动画原则

```
- 有目的: 每个动画都应服务于用户体验
- 有节制: 动画时长不超过 500ms (复杂场景除外)
- 有反馈: 用户操作必须有即时视觉反馈
- 有性能: 使用 transform 和 opacity，避免触发重排
```

### 7.2 缓动函数 (Easing)

| 名称 | 值 | 用途 |
|------|-----|------|
| ease-default | `cubic-bezier(0.22, 1, 0.36, 1)` | 默认过渡，自然减速 |
| ease-bounce | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 弹性效果，按钮弹出 |
| ease-smooth | `cubic-bezier(0.4, 0, 0.2, 1)` | 平滑过渡，颜色变化 |
| ease-in-out | `cubic-bezier(0.65, 0, 0.35, 1)` | 对称缓动，页面切换 |
| linear | `linear` | 匀速，轮播、进度条 |

### 7.3 动画时长

| 类型 | 时长 | 用途 |
|------|------|------|
| instant | 100ms | 颜色变化、状态切换 |
| fast | 200ms | 按钮悬停、链接下划线 |
| normal | 300ms | 卡片悬停、下拉展开 |
| slow | 400ms | 模态框、灯箱 |
| entrance | 600ms | 页面入场、滚动显示 |
| hero | 800ms | Hero轮播、大图切换 |

### 7.4 页面过渡动画

```
路由切换:
- 当前页面: opacity 1 → 0, 200ms
- 新页面: opacity 0 → 1, translateY 20px → 0, 300ms
- 缓动: ease-default

滚动显示 (IntersectionObserver):
- 触发: 元素进入视口 20%
- 动画: opacity 0 → 1, translateY 40px → 0
- 时长: 600ms
- 缓动: ease-default
- 交错: 多个元素间隔 100ms
```

### 7.5 微交互

#### 7.5.1 按钮点击

```
按下: scale 1 → 0.97, 100ms, ease-smooth
释放: scale 0.97 → 1, 200ms, ease-bounce
```

#### 7.5.2 卡片悬停

```
悬停进入:
- translateY: 0 → -4px
- shadow: 无 → 0 8px 24px rgba(0,0,0,0.4)
- border-color: #2a2a2a → #3a3a3a
- duration: 300ms
- easing: ease-default

悬停离开:
- 反向动画
- duration: 200ms
```

#### 7.5.3 图片加载

```
骨架屏 → 模糊渐进加载 → 清晰图片

阶段1: 骨架屏 (灰色背景 #1a1a1a)
阶段2: 模糊缩略图 (filter: blur(20px), opacity 0.5)
阶段3: 清晰图片 (filter: blur(0), opacity 1)

过渡: opacity 0.5s, filter 0.5s
```

#### 7.5.4 搜索框聚焦

```
聚焦:
- border-color: #2a2a2a → #f5c518
- shadow: 无 → 0 0 0 3px rgba(245,197,24,0.15)
- 搜索图标: #666666 → #f5c518
- duration: 200ms
```

#### 7.5.5 轮播切换

```
自动轮播:
- 间隔: 5000ms
- 切换动画: opacity 0 → 1, 800ms
- 图片缩放: scale 1.05 → 1, 1200ms
- 缓动: ease-in-out

手动切换:
- 动画时长: 400ms
- 方向: 根据切换方向决定滑动方向
```

### 7.6 性能优化

```
- 使用 transform 和 opacity 进行动画
- 避免动画 width、height、top、left
- 使用 will-change 提示浏览器优化
- 动画结束后移除 will-change
- 支持 prefers-reduced-motion 媒体查询
```
